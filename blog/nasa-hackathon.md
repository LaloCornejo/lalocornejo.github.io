# NASA Hackathon — Top 6 Nationally

**Date:** 2025-10-27

**Tags:** nasa, hackathon, threejs, space

---

I participated in a NASA hackathon and our project placed **Top 6 in the National scoring**. The project was a 3D Earth-Moon-NEO visualization system built with Three.js.

## The Project

The application renders the Earth-Moon system with real Near-Earth Objects (NEOs) using actual NASA data. It features:

- **Real orbital mechanics** using the Vis-Viva equation
- **Live data** from NASA's NEO API
- **Interactive 3D** with orbit, pan, and zoom controls
- **Impact assessment** calculations with comparative scaling
- **Glassmorphism UI** inspired by Apple Vision Pro

## Technical Details

Built with vanilla JavaScript, Three.js, and Python for data processing. The orbital mechanics implementation calculates elliptical orbits with inclination and eccentricity for each NEO.

### What I learned

- WebGL rendering optimization (LOD, frustum culling, geometry pooling)
- REST API integration with NASA's NeoWs service
- Mathematical modeling of celestial mechanics
- Presentation and pitching skills for technical projects
